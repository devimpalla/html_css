Position

  • fixed : 브라우저를 기준으로 특정 위치에 고정으로 배치
  • static (default) : 정상적인 흐름(위치)
  • relative : 정상적인 위치로 부터 상대적인 태그(요소-element)를 배치
  • absolute : 부모태그가 relative일때, 혹은 부모 태그 중 relative가 없으면
    전체페이지(body)를 기준으로 시작위치에서 top, left, bottom, right 등의 절대적 수치만큼 배치
This div element has position: relative;
This div element has position: absolute;
