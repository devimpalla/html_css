Javascript Data Type

  • 숫자 : Number (정수, 실수)
  • 문자 : String (따옴표(',")로 감싼 모든 것)
  • 참거짓 : boolean(true, false)
  • 객체 : object
  • 배열 : Array (배열객체)
  • 미정된 값 : undefined
  • 알 수 없는 존재 : null

Data type을 알고 싶을 때 사용하는 키워드 : typeof

자바스크립트의 기본 데이터 타입: Primitive Type

  • number
  • boolean
  • string(원래는 참조형 데이터 타입인데 특수하게 변형된 형태로 변경해서 사용)
  • 다른 프로그래밍 언어에서는 string은 참조형 데이터 타입임

자바스크립트의 참조 데이터 타입: Reference Type

  • string
  • Object
  • Array
  • function
  • etc...
